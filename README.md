# Oxygen
Project to explain Design Pattern Flyweight.

## Getting Started

### Prerequisites

JDK 6 installed

### Installing

git clone https://github.com/sidneynakatani/oxygen.git

cd oxygen

import in your IDE (Eclipse/IntelliJ)

run main Application
